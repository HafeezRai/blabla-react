## Frontend for Blablacar clone application using React + Redux + [*Rails API*](github.com/marcinkornek/blabla-clone-api)

#### Description
Application which uses React + Redux + [*Rails API*](github.com/marcinkornek/blabla-clone-api).

## Setup
Install packages:
```
$ npm install
```

## Usage
Run server:
```
$ npm run start-dev
```
